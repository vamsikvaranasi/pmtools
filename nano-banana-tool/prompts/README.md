# Prompt Library

This directory contains reusable prompt templates for image generation.

## Adding Prompts

1. Create a new `.txt` or `.md` file in this directory
2. Use a descriptive filename (e.g., `product-mockup.txt`, `social-media-post.md`)
3. Write your prompt text in the file (one prompt per file)
4. The prompt will automatically appear in the interactive mode prompt selection menu

## File Format

- One prompt per file
- Plain text format (`.txt` or `.md` extension)
- No special formatting required
- The entire file content will be used as the prompt
